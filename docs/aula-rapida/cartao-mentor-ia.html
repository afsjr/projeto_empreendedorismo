💡 DICAS IMPORTANTES

  • Não peça para o Mentor fazer tudo! Ele é um assistente, não faz o trabalho por vocês.
  • Use o Mentor para inspirar ideias — não para copiar respostas.
  • Compare: O que o Mentor sugeriu com o que o grupo já mapeou.
  • Surpresa: Se o Mentor sugerir algo que vocês não pensaram, anotem na Missão Bônus!
  • Tempo: Vocês têm 15 minutos — usem o Mentor com sabedoria!
