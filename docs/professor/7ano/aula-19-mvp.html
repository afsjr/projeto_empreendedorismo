🎯 FECHAMENTO

Pergunta: Por que começar pequeno?

Resposta: Validar ideia antes de investir

Tarefa: Criar MVP em casa

💡 ORIENTAÇÕES PEDAGÓGICAS

  1. Simplicidade: Menos é mais no início. Reforce que o MVP não precisa ser perfeito.
  2. Feedback rápido: Testar cedo e frequentemente traz aprendizado real.
  3. Prototipação: O protótipo em papel é suficiente para começo. Não precisa de tecnologia.
  4. Foco: Ajude os alunos a identificar a funcionalidade mais essencial do seu produto.
  5. Validação: Ensine que o objetivo do MVP é aprender, não ganhar dinheiro imediatamente.

📝 NOTAS DE AVALIAÇÃO

  • Observe se os alunos conseguem identificar o essencial para o MVP.
  • Avalie a qualidade dos protótipos em papel criados.
  • Verifique se o teste com colega gerou aprendizado real.
  • Prepare os alunos para criar protótipos no projeto Startup Mirim.
