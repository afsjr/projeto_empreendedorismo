🎯 FECHAMENTO

Resumo: Você consegue criar um orçamento?

Próximo: Projeto de consultoria com famílias reais

Tarefa: Conversar com a família sobre orçamento

💡 ORIENTAÇÕES PEDAGÓGICAS

  1. Privacidade: Não force o compartilhamento de valores pessoais das famílias dos alunos.
  2. Exemplo: Use valores hipotéticos se algum aluno se sentir desconfortável.
  3. Regra 50-30-20: Explique: 50% necessidades, 30% desejos, 20% poupança/investimentos.
  4. Dificuldades esperadas: Alunos podem ter dificuldade em separar despesas fixas de variáveis. Reforce a diferença.
  5. Inclusão: Adapte os valores dos exemplos para a realidade local da comunidade.

📝 NOTAS DE AVALIAÇÃO

  • Observe se os alunos conseguem classificar corretamente as despesas.
  • Avalie se o cálculo do saldo final está correto.
  • Verifique se as sugestões de economia são realistas e aplicáveis.
  • Prepare os alunos para aplicar esse conhecimento no projeto semestral.

🔗 PROJETO DO SEMESTRE

No Consultoria Financeira Mirim, os grupos farão análise real com famílias, criando orçamentos completos e dando sugestões de melhoria baseadas na regra 50-30-20.
