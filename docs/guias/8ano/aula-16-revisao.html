❓ Perguntas Provocadoras (Exemplos para o Quiz)

P: Qual a diferença entre custo fixo e variável?
R: Fixo não muda com quantidade vendida (aluguel). Variável muda conforme produção (matéria-prima).
P: O que é Markup?
R: É a porcentagem adicionada ao custo para chegar ao preço de venda. Ex: custo R$10 + 50% markup = preço R$15.
P: Como calcular ponto de equilíbrio?
R: PE = Custos Fixos ÷ (Preço - Custo Variável). É a quantidade mínima para não ter prejuízo.
