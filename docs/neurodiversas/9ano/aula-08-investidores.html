🧠 Tipos de investidores

1

Anjo

Pessoa física. Investe com próprio dinheiro. Primeiros estágios.

2

VC (Venture capital)

Fundos profissionais. Investem muito dinheiro. Já validates.

3

Accelerators

Aceleradoras. Dinheiro + mentoria. Ex: Y Combinator

🎯 Escolher quando

  • Anjo = ideia + produto
  • VC = crescimento
  • Accelerator = estrutura

💡 Dica TDAH

Build relationships early! Investors take time to find.

🤝 Dica TEA

Cada investor has specific criteria. Clear rules help!
