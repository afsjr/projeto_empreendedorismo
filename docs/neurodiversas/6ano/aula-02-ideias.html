Como Gerar Ideias

⏱️ Tempo: 10 min | 🧠 Aula Curta

📌 O que você vai aprender

  • OBJETIVOAprender técnicas para criar ideias
  • Descobrir como pensar em soluções
  • Criar sua primeira ideia

🧠 Ideia boa = Problema + Solução

😱 PROBLEMA → 💡 IDEIA → ✅ SOLUÇÃO

1

Olhe ao redor

O que você vê que pode melhorar?

2

Pergunte "e se...?"

E se existisse uma forma diferente?

3

Misture ideias

Combine duas coisas diferentes

⏸️🧠 Pausa! Espaço para pensar

Faça uma pausa de 30 segundos. Respire. Depois continue.

💡 Dica para TDAH

Anote suas ideias mesmo que parecerem bobas. Às vezes a ideia "maluca" é a melhor!

🤝 Dica para TEA

Não precisa ter ideia "perfeita". Pode começar com algo simples e melhorar depois.

✅ Resumo

  • Ideia = problema + solução
  • Pergunte "e se...?"
  • Anote todas as ideias
